## ✅ РЕШЕНИЕ:

### 1️⃣ Переключить сервис бота на Docker

**НА RENDER:**

1. Зайдите на https://dashboard.render.com
2. Откройте сервис **bot**
3. Нажмите **Settings** (слева)
4. Найдите раздел **Environment**
5. Нажмите **Edit**
6. Выберите **Docker** вместо **Python**
7. **Docker Command:** оставьте пустым (будет использован CMD из Dockerfile)
8. Нажмите **Save Changes**
9. Нажмите **Manual Deploy** → **Clear build cache & deploy**

**РЕЗУЛЬТАТ:** Бот задеплоится через Docker с Python 3.11
