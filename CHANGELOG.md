## 🚀 Как запустить

### 1. Запуск бота локально:

```bash
# Активируйте виртуальное окружение
.venv1\Scripts\activate

# Запустите бота
python main.py
```

### 2. Запуск с ngrok (для тестирования Mini App):

```bash
# В отдельном терминале запустите ngrok
ngrok http 8080

# Скопируйте HTTPS URL из ngrok
# Обновите WEBAPP_URL в keyboards.py
```

### 3. Тестирование:

1. Откройте бота в Telegram
2. Нажмите /start
3. Нажмите кнопку "🎁 Открыть кейсы"
4. Проверьте:
   - ✅ Отображается ваше реальное имя
   - ✅ Баланс сохраняется после перезапуска
   - ✅ Интерфейс адаптируется под размер экрана
   - ✅ Все кнопки работают корректно

## 📊 Структура базы данных

Все данные пользователей сохраняются в `bot_users.db`:

- **users** - основная информация о пользователях
- **user_balance** - баланс пользователей
- **user_game_data** - игровые данные (уровень, опыт, энергия)
- **user_upgrades** - купленные улучшения
- **user_achievements** - достижения
- **user_inventory** - инвентарь предметов
- **case_openings** - история открытий кейсов

## 🔧 Настройка

### Изменить URL Mini App:

Отредактируйте `keyboards.py`:

```python
WEBAPP_URL = "https://your-domain.com/webapp/index.html"
```

### Изменить ID администратора:

Отредактируйте `main.py`:

```python
ADMIN_ID = 1121167993  # Ваш Telegram ID
```

## 🆘 Решение проблем

### Проблема: Не отображается имя пользователя
**Решение:** Проверьте, что пользователь написал боту /start перед открытием Mini App

### Проблема: Баланс сбрасывается
**Решение:** Убедитесь, что веб-сервер запущен и доступен (порт 8080)

### Проблема: Mini App не открывается
**Решение:** 
1. Проверьте, что WEBAPP_URL использует HTTPS
2. Убедитесь, что ngrok запущен
3. Обновите URL в keyboards.py

### Проблема: Данные не сохраняются
**Решение:**
1. Проверьте логи в консоли браузера (F12)
2. Убедитесь, что API endpoints доступны
3. Проверьте файл bot_users.db

## 📞 Поддержка

Если возникли проблемы:
