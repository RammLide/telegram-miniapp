# Telegram Bot с Mini App для открытия кейсов v3.0

Современный телеграм бот с системой управления админами, расширенной статистикой и встроенным Mini App для открытия кейсов.

## Возможности

### 🤖 Основные функции бота:
- Автоматическая регистрация пользователей
- 📢 Рассылка сообщений всем пользователям
- 📊 Детальная статистика и аналитика
- 👥 Просмотр списка пользователей с пагинацией
- 👨‍💼 Система управления администраторами
- 🎨 Красивый интерфейс с кнопками

### 🎁 Mini App - Открытие кейсов:
- 4 типа кейсов (Бронзовый, Серебряный, Золотой, Платиновый)
- Система редкости предметов (Common, Rare, Epic, Legendary)
- Инвентарь пользователя
- Баланс и экономика
- Анимация открытия кейсов
- Статистика открытий

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

2. Создайте файл `.env`:
```
BOT_TOKEN=ваш_токен_от_BotFather
```

3. В файле `main.py` замените `ADMIN_ID` на ваш Telegram ID (строка 43)

4. В файле `keyboards.py` замените URL для Mini App на ваш домен (строка 8):
```python
web_app=WebAppInfo(url="https://your-domain.com/webapp/index.html")
```

## Запуск

```bash
python main.py
```

Бот запустится на порту 8080 для веб-сервера Mini App.

## Настройка Mini App

### Для локального тестирования:
1. Используйте ngrok или аналогичный сервис для создания публичного URL:
```bash
ngrok http 8080
```

2. Замените URL в `keyboards.py` на полученный от ngrok

### Для продакшена:
1. Разместите файлы из папки `webapp/` на вашем веб-сервере
2. Убедитесь, что сервер поддерживает HTTPS
3. Обновите URL в `keyboards.py`

## API Endpoints

- `POST /api/balance` - Получение баланса пользователя
- `POST /api/open_case` - Открытие кейса
- `POST /api/inventory` - Получение инвентаря
- `POST /api/stats` - Статистика открытий

## База данных

### Таблицы:
- `users` - Пользователи бота
- `admins` - Администраторы
- `statistics` - Статистика событий
- `user_balance` - Баланс пользователей
- `user_inventory` - Инвентарь пользователей
- `case_openings` - История открытий кейсов
